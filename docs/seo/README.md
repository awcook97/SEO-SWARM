# SEO documentation index

This index explains what each document is for and when to use it. Use it alongside the main workflow in @docs/seo/swarm-execution-workflow.md.

## Measurement

- Measurement spec: @docs/seo/measurement-spec.md  
  Defines KPIs, metrics, and how to interpret results.
- Measurement intake template: @docs/seo/measurement-intake-template.md  
  Collects the source data you need before generating reports.
- Measurement reporting template: @docs/seo/measurement-reporting-template.md  
  Formats the final report output for client delivery.
