# Rank tracking and competitor analysis spec (reusable)

This spec defines the minimum required inputs, cadence, outputs, and quality checks for rank tracking and competitor analysis. It is client-agnostic and should be used across all local SEO engagements.

## Required inputs

- Client name: [Client Name]
- Market/location scope: [City/Region], [Service Areas]
- Primary services: [Service List]
- Keyword set: [Primary Keywords], [Secondary Keywords], [Branded Keywords]
- Target URLs: [Service Pages], [Location Pages], [Content Hubs]
- Competitor set: [Primary Competitors], [Secondary Competitors]
- SERP feature targets: [Local Pack], [Maps], [FAQ], [Reviews], [Sitelinks]
- Tracking cadence: [Weekly] or [Monthly]
- Source systems: [Rank tracker], [GSC], [GBP], [Analytics]

## Rank tracking requirements

- Track rankings for each keyword by location.
- Track both organic and local pack where available.
- Record current rank, best rank, and change vs last period.
- Flag any keyword with a drop of [X] positions.
- Record the URL ranking for the keyword (must match target URL list).

## Competitor analysis requirements

- Record competitor domain, name, and primary services.
- Capture top 3 ranking pages per primary keyword.
- Note SERP features and presence (local pack, reviews, FAQs).
- Identify content gaps: missing services, missing service areas, missing FAQs.
- Note differentiators: reviews count, star rating, unique offers (verified only).

## Output artifacts

- Rank tracking report (CSV + summary).
- Competitor snapshot (table + notes).
- Insights summary: top gains, top drops, quick wins.
- Recommendations list aligned to service priorities.

## Data quality and compliance

- Do not fabricate competitor data; only include verified sources.
- Use consistent NAP for client references in reports.
- Keep sources and export timestamps in every report.
- Avoid claims like “#1” unless backed by dated evidence.

## Update cadence

- Weekly: quick summary + alerts for drops.
- Monthly: full report + recommendations.

## Integration touchpoints

- Templates: include rank tracking and competitor inputs in page/article briefs.
- Reporting: align insights with monthly performance reports.
